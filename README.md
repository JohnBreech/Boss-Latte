# ☕ Boss Latte

A Google Apps Script automation that runs a full **weekly strategy review** every Sunday at 6pm — pulling your real Calendar and Tasks data, sending it to Gemini for an honest executive-style breakdown, emailing you the report, and logging your scores to a Google Sheet with an auto-updating trend chart.

Boss Latte doesn't reward busyness. It checks whether your week actually moved you closer to your long-term goals — academics, freelance/career, software projects, learning, finances, health, relationships, and rest — and tells you the truth about it.

Built as a companion automation to [Axile](#), my Gemini-powered executive assistant.

---

## What it does

Every Sunday at 6pm, the script:

1. Pulls the past 7 days and next 7 days from Google Calendar
2. Pulls completed and incomplete Google Tasks
3. Pulls your last 4 weeks of scores from the Sheet for trend context
4. Sends everything to Gemini with a structured weekly-review prompt
5. Emails you the full narrative report (scorecard, goal alignment by area, time audit, wins, bottlenecks, project health, next week's priorities, suggested schedule, risks, and a coach's closing note)
6. Appends a row of scores to a **Weekly Log** Google Sheet
7. Refreshes a line chart of your Overall Score trend, right in the Sheet

## Setup

### 1. Create the Google Sheet
Create a blank spreadsheet at [sheets.google.com](https://sheets.google.com). Copy its ID from the URL (the string between `/d/` and `/edit`). Leave it empty — the script creates its own "Weekly Log" tab and headers on first run.

### 2. Get a Gemini API key
Grab one from [Google AI Studio](https://aistudio.google.com/app/apikey).

### 3. Configure the script
Open the project in the [Apps Script editor](https://script.google.com) (or push via `clasp`, see below), and:

- In `BossLatte.gs`, update:
  ```javascript
  const SHEET_ID = 'your-sheet-id-here';
  const MY_EMAIL = 'your-email@example.com';
  ```
- Go to **Project Settings → Script Properties** and add:
  | Property | Value |
  |---|---|
  | `GEMINI_API_KEY` | your Gemini API key |

### 4. Enable the Tasks API service
In the Apps Script editor: **Services (+) → Tasks API → Add**.

### 5. Install the trigger
Run `setupTrigger()` once manually from the editor. This installs the recurring Sunday 6pm trigger — you only need to do this once.

### 6. Test it
Run `runBossLatte()` once manually. Check your inbox for the report and confirm the Sheet's "Weekly Log" tab populated correctly before trusting the automation.

## Developing locally with clasp

This project can be edited locally and synced to Apps Script using [`clasp`](https://github.com/google/clasp).

```bash
npm install -g @google/clasp
clasp login
clasp clone <your-script-id>   # or: clasp create --title "Boss Latte" --type standalone
```

After editing:
```bash
clasp push   # push local changes to Apps Script
clasp pull   # pull remote changes back down
```

> `.clasprc.json` (your personal auth token) is gitignored and should never be committed.

## Notes

- Score history for trend context is read from the last 4 rows of the "Weekly Log" sheet — no separate database needed.
- The chart only appears once there are 2+ weeks of data logged.
- Secrets (API keys) live in Script Properties, never in code — safe to make this repo public if you want, though it's private by default.
